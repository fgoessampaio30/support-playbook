# 🧠 Support Playbook

Repositório de conhecimento técnico voltado à análise de incidentes, troubleshooting e investigação de problemas em sistemas corporativos.

---

## 🎯 Objetivo

Documentar práticas reais de análise de incidentes, incluindo:

- Investigação de erros em banco de dados Oracle
- Análise de performance e SQL
- Interpretação de logs de aplicação
- Análise de tráfego HTTP (HAR)
- Diagnóstico de integrações e APIs
- Estruturação de Root Cause Analysis (RCA)

---

## 🧩 Estrutura do repositório

### 📁 Oracle
Análise de erros e investigação de causa raiz em banco de dados.

### 📁 SQL
Consultas e técnicas de diagnóstico de problemas em banco de dados.

### 📁 HAR
Análise de requisições HTTP para troubleshooting de aplicações web.

### 📁 Logs
Interpretação de logs de aplicação e servidor.

### 📁 RCA
Metodologia estruturada de análise de causa raiz.

---

## ⚙️ Propósito

Este repositório não foca em exemplos acadêmicos ou exercícios de programação.

Ele foca em **cenários reais de suporte e sustentação de sistemas**, com abordagem prática de investigação de incidentes.

---

## 📌 Nota

Todo conteúdo é baseado em experiências reais de análise de incidentes, com dados anonimizados e sem qualquer informação sensível de sistemas ou clientes.
